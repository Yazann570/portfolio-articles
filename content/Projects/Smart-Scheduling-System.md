# Smart Scheduling System

An AI-powered academic scheduling platform developed as my graduation project. The system helps university students generate multiple conflict-free schedules, compare them, and receive AI-assisted recommendations for the best schedule.

## Overview

Course registration is often a frustrating process because students must manually compare sections, instructors, and schedules while avoiding conflicts. This project automates that process by generating valid schedules and ranking them using Google's Gemini API.

## Features

- Generate multiple valid schedules automatically
- Detect scheduling conflicts
- Filter schedules by instructor or section
- Save favorite schedules
- AI-generated ranking and recommendations using Gemini API
- Student-friendly interface
