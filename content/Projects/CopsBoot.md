# CopsBoot

CopsBoot is a police reporting system consisting of a Kotlin Spring Boot backend and an Android client application.

## Overview

The project demonstrates how a mobile application communicates with a RESTful backend to manage police reports and related data.

## Features

- RESTful API
- Authentication
- CRUD operations
- Database integration with JPA
- Android client
- Image upload support

## Tech Stack

- Kotlin
- Spring Boot
- Spring Data JPA
- H2 Database
- Android
- Gradle

## My Contributions

- Developed backend APIs
- Implemented authentication
- Connected Android app to backend
- Designed database entities
- Built CRUD functionality

## Challenges

Learning Kotlin alongside Spring Boot while designing a clean API architecture was the biggest challenge.

## What I Learned

- Spring Boot development
- Kotlin language
- REST API design
- Android and backend communication
- JPA and ORM concepts
