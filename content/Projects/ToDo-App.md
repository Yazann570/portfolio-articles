# To Do List App

A lightweight task management application built to practice React fundamentals.

## Overview

Users can create, manage, and organize daily tasks through a clean and simple interface.

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Simple responsive interface

## Tech Stack

- React
- JavaScript
- HTML
- CSS

## What I Learned

- Component-based development
- React state management
- User interaction
- Building responsive interfaces
